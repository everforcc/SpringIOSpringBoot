### 安装 Gem

~~~
1. 下载后解压，当前目录切换到解压目录中，如 D:\software\redis\rubygems-2.6.12 （即redis的目录）
2. 然后在命令行执行 ruby setup.rb。
3. 切换到redis目录执行 gem install redis
~~~
