- 命令

1. Psubscribe 命令
- 订阅一个或多个符合给定模式的频道。
- 语法
~~~
redis 127.0.0.1:6379> PSUBSCRIBE pattern [pattern ...]
支持正则语法
~~~
- 版本 >= 2.0.0
- 返回值 接收到的消息
~~~

~~~

2. Pubsub 命令
- 查看订阅与发布系统状态。
- 语法
~~~
redis 127.0.0.1:6379> PUBSUB <subcommand> [argument [argument ...]]
~~~
- 版本 >= 2.8.0
- 返回值 由活跃频道组成的列表。
- 

3. Publish 命令
- 将信息发送到指定的频道。
- 语法
~~~
redis 127.0.0.1:6379> PUBLISH channel message
~~~
- 版本 >= 2.0.0
- 返回值 接收到信息的订阅者数量。
-
