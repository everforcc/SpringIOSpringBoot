###  常用方法

~~~

add( )         向集合中添加元素

clear( )        去掉集合中所有的元素

contains( )    判断集合中是否包含某一个元素

isEmpty( )    判断集合是否为空

iterator( )    主要用于递归集合，返回一个Iterator()对象

remove( )    从集合中去掉特定的对象

size( )        返回集合的大小
~~~
