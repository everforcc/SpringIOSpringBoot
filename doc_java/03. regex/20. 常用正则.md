### 语言字符

~~~
中文字符
[u4E00-u9FA5]
全角字符
?[uFE30-uFFA0]
匹配双字节字符(包括汉字在内)：
[^x00-xff]
- 可以用这个来转换计算字符串的长度，一个汉字两位
~~~

### 常见业务，电话等各种号码

- 匹配国内电话号码
~~~
d{3}-d{8}|d{4}-d{7} 
0511-4405222 或 021-87888822
~~~
- qq号，腾讯QQ号从10000开始
~~~
[1-9][0-9]{4,} 
~~~
- 邮政编码为6位数字
~~~
[1-9]d{5}(?!d) 
~~~
- 匹配身份证,身份证为15位或18位
~~~
d{15}|d{18} 
~~~
- ip
~~~
d+.d+.d+.d+
~~~
- 匹配URL
~~~
[a-zA-z]+://[^s]* 
通用的，http和ftp等没有分开
~~~
- 邮箱
~~~
w+([-+.]w+)*@w+([-.]w+)*.w+([-.]w+)* 
~~~
- 用户名包含字符、数字、下划线和连接字符(-)，并设置用户名的长度
~~~
^[a-z0-9_-]{3,15}$
~~~
- 密码
~~~
# 至少一个大写，小写，数字，字符 
String pattern = "^(?![^a-z]+$)(?![^A-Z]+$)(?!\\D+$)(?![a-zA-Z0-9]+$).{8,}$";
~~~

### 语言格式

- 匹配简单的HTML
~~~
<(S*?)[^>]*>.*?|<.*? /> 
仅仅能匹配部分，对于复杂的嵌套标记依旧无能为力
~~~
- 匹配HTML标记的正则表达式：
~~~
/<(.\*)>.\*</1>|<(.*) />/
~~~
- 匹配首尾空白字符
~~~
~~^s*|s*$~~ 
用来删除行首行尾的空白字符(包括空格、制表符、换页符等等)，非常有用的表达式  
~~~
- 匹配空行的正则表达式：
~~~
n[s| ]*r
~~~
- 匹配首尾空格的正则表达式：
~~~
(^s\*)|(s*$)
~~~
