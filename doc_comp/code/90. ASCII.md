ASCII（American Standard Code for Information Interchange，美国信息交换标准代码）是基于拉丁字母的一套电脑编码系统，主要用于显示现代英语和其他西欧语言。ASCII 码使用指定的7 位或8 位二进制数组合来表示128 或256 种可能的字符。
