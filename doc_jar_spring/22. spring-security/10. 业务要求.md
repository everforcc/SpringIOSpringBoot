登录url:/login
支持三种登录传参
### 登录： JSON 模式
    POST http://{{host}}:{{port}}/login
     Content-Type: application/json
     {"username": "admin","password": "superadmin"}
###
### 登录：表单模式
    POST http://{{host}}:{{port}}/login
     Content-Type: application/x-www-form-urlencoded
     username=admin&password=superadmin
###
### Basic 模式，无 session
    GET http://{{host}}:{{port}}/user
    Content-Type: application/json
    Authorization: basic YWRtaW46c3VwZXJhZG1pbg==
###
