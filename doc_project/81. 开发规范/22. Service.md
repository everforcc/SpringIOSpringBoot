## service

### 1. 必须传入对象，禁止传入字符串等数据

- 因为对象可以用Validated来对数据进行校验

### 2. 数据校验

~~~
@Validated
@Transactional
public class WebSiteService {
    @Validated({ISave.class})
    @Transactional(rollbackFor = Exception.class)
    public WebSiteDto insert(@Valid WebSiteDto webSiteDto){
        return webSiteDto;
    }
}    
~~~
