- 分支

- 查看分支
~~~
# 当前分支
git branch
# 远程分支
git branch -r
~~~
- 创建新分支
~~~
git checkout -b 分支名;  #创建分支
git push --set-upstream origin 分支名;  #提交分支到远程仓库

#例如创建远程分支master_branch
git checkout -b master_branch;
git push --set-upstream origin master_branch;
~~~
- 查看远程分支的提交
~~~
git branch -r -v;
~~~
